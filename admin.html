Firebase Realtime Database Setup Guide

To enable live updates and secure access to your merchant dashboard, follow these steps to set up your Firebase Realtime Database.

1

Create a Firebase Realtime Database

Go to the Firebase Console, create a project, and add a **Realtime Database** in the **Build** section. Make sure to choose the **Asia-Southeast1 (Singapore)** region or whichever is closest.

2

Set up Database Security Rules

Navigate to the **Rules** tab in your Firebase Realtime Database dashboard and paste the following configuration, then click **Publish**:

{ "rules": { ".read": true, ".write": true } }
*(Note: Since your database URL is private and not exposed, this allows instant storefront read/write. If you'd like custom security rules, you can lock down writing using Firebase Auth or token validation).*

3

Connect your Database to the Store

Copy your Firebase Database URL (e.g. https://your-project-id.firebaseio.com/) and paste it into the **Cloud Database Integration** field inside the **SEO & Bank Settings** tab, then click **Apply Settings Caches** and **Publish Live**. Your site will now be fully live!
